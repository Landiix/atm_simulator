# atm_simulator
Симулятор банкомата
---
 Описание проекта
 сервис позволяет имитировать поведение банкомата с выполнением базовых операций ведения счета.
 Поддерживаемые операции:
 - просмотр баланса счета
 - пополнение счета
 - перевод средств между счетами
 - открытие вклада
 - начисление процентов с течением времени
